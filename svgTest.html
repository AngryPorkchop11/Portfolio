About Me

Hi, my name is Mathew Lister. I just graduated from Seattle Pacific University with a B.A. in Computer Science. While learning to code, I became excited about web development and its limitless capabilities. I enjoy building software in teams that encourage outside of the box ideas.


I can work with server code plus; I have a sense for UI/UX. I believe everyone should be able to use the web without discrimination. Because of that, I always strive to improve my responsive web development skills. I'm very comfortable using Git and the Linux CLI. I have developed software in teams following the Agile methodology using Jira for our scrum board. I'm actively seeking an entry-level programming position.
