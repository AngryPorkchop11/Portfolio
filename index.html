Music Database
(A Mock-Spotify)

SQL database with Java interface.

A three person group project inspired by spotify and itunes. This project gives users the ability to manage playlists, concerts, songs, albums, artists and more. The interface makes our database searchable and modifyable. The scope of this project is intended to show backend capabilities with less emphasis on front end.

Advanced Techniques

Cache and Hash

When inserting a song from the database into a playlist we only wanted to use ID numbers. However, for the user we wanted them to be able to search with song names. We cached the results from the SQL query with songs related to what the user searched. We used a hashmap inorder to creat unique value pairs based on song and artist name, with the ID number associated with the song as the value for quick and reliable song retrival.
